## Getting Started

Installation
- Clone the repository
    ```
        git clone https://github.com/maddieking02/clubspot-takehome.git
    ```
- Install the dependencies
    ```
        npm i
    ```
- Run the following script
    ```
        npm run react-dev
    ```
